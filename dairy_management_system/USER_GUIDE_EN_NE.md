# Dairy Management System User Guide / प्रयोगकर्ता निर्देशिका

## Admin Guide English

1. Login as admin.
2. Open Dairy Profile and enter dairy name, address, PAN, contact, logo, bill header, and bill footer.
3. Add shops / branches.
4. Add units such as kg, litre, gram, ml, packet, and piece.
5. Add products such as milk, curd, cheese, ghee, paneer, butter, cream, and yogurt.
6. Enter Daily Product Receiving; the system creates batch numbers automatically.
7. Add sellers and customers.
8. Add payment methods.
9. Use Bill Format to manage bill header/footer style.
10. Use Sales and MIS reports for daily, product-wise, batch-wise, customer-wise, seller-wise, shop-wise, payment, stock, low stock, expired product, monthly, profit/loss, and due payment reports.
11. Use Activity Logs to track system changes.
12. Use Backup / Restore to download, upload, and restore database backups.

## Seller Guide English

1. Login as seller.
2. Check dashboard for today’s sales and available batches.
3. Search product by name, scan code, or batch number.
4. Add or select customer.
5. Select shop, product batch, quantity, payment method, and discount if needed.
6. Generate bill.
7. Print or download bill.
8. Stock is automatically deducted after billing.

## Seller Guide Nepali

1. Seller को रूपमा लगइन गर्नुहोस्।
2. Dashboard मा आजको बिक्री र उपलब्ध batch हेर्नुहोस्।
3. Product name, scan code वा batch number बाट खोज्नुहोस्।
4. Customer छान्नुहोस् वा नयाँ customer दर्ता गर्नुहोस्।
5. Shop, product batch, quantity, payment method र discount छान्नुहोस्।
6. Bill generate गर्नुहोस्।
7. Bill print वा download गर्नुहोस्।
8. Billing पछि stock स्वतः घट्छ।

## Customer Guide English

1. Login from website or mobile app.
2. View profile.
3. Update profile.
4. View products.
5. View purchase history.
6. View bills and payment status.
7. Download invoice.
8. View notifications.
9. Change password.
